## 分页查询 LIMIT

### 1. 语法

```sql
SELECT 字段列表 FROM 表明 LIMIT 起始索引, 查询记录数;
```

### 2. 注意

- 起始索引从 0 开始，起始索引 = （查询页码 - 1） \* 每页记录数
- 分页查询是数据库的方言，不同数据库有不同的实现，MySQL 中是 LIMIT
- 如果查询的是第一页数据，起始索引可以省略，直接简写为 LIMIT 10

### 3. 示例

- 查询第1页员工数据，每页展示 10 条记录

```sql
select * form emp limit 10;
```

- 查询第2页员工数据，每条展示 10 条记录
> （页码-1）\*页面展示记录数

```sql
select * from emp limit 10, 10;
```
