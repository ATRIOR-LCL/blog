## 一、
- `T`：CPU 有效访问时间
- `t1`：访问快表的时间
- `t2`：访问内存的时间
- `p`：概率

`T = t1 + (2 - p) * t2`

## 二、
- `r`：每个进程需要的资源
- `R`：总资源
- `P`：进程数

发生死锁：`(r - 1) * P + 1 > R`
不发生死锁：`(r - 1) * P + 1 <= R`

## 三、
- `A`：逻辑地址
- `p`：页号
- `d`：页内地址
- `s`：页面大小

`s * p + d = A`

## 四、

- `c`：簇数
- `C`：簇大小
- `G`：容量

`c * 8 * pow(C,2) = G`

## 五、

- `p`：页面大小
- `s`：页表项大小
- `l`：页表页长
- `A`：逻辑地址
- `n`：页表项个数

`p / s = l`

`A / l = n`

## 六、

- `u`：频率
- `v`：速率
- `b`：位数
- `t`：平均响应时间

`u = v / b`

单缓冲：`t = 1 / v`

双缓冲：`t = b / v`

## 七、

- `C`：磁盘容量
- `n`：磁道数
- `r`：转速
- `v`：传输速率

`C / n * r = v`
